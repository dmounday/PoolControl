Simple Pool Equipment Control
Features:
	Main Pump control
	Booster Pump 
	Sanitizer on/off
	Heat Pump on/off temperature set.


Generalized configuration of sensors and equpment.
Written in C++17 

To build see the README in the PoolControllRPi directory.
